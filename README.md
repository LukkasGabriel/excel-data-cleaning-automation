# Automated Excel Data Cleaning & Reporting

Python automation that cleans, validates and structures complex Excel datasets.  
The script processes multiple sheets, removes duplicates, standardizes text fields, converts data types and recalculates revenue.

It then generates automated summaries and pivot-style reports using Pandas groupby operations and merges.

The final Excel file exports clean tables, regional revenue summaries and formatted financial values, transforming messy spreadsheets into reliable business reports.

---

# Features

• Load and process multiple Excel sheets automatically  
• Remove invalid rows and duplicated orders  
• Standardize text fields (Region, Seller, Product, Category)  
• Convert and validate data types  
• Recalculate revenue values  
• Generate revenue summaries by region  
• Simulate pivot-style reports using Pandas  
• Export a clean and structured Excel report
